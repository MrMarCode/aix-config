# Task Loop

Periodically look at the file todos.md as your main task list. It may be updated or changed as I see fit so it's important to check it regularly.

Handle each of the bullet points and don't stop until you've accomplished all your tasks. NEVER STOP even after you've finished all tasks. Come back and ask me with the ask questions tool what I'd like you to do after that. Get familiar with the codebase and plan things thoroughly. Test things thoroughly with the playwright MCP tool or tauri mcp tool (if this is a tauri app or web app). If you encounter issues you can't recover from, use the question ask tool to ask me for help, but don't end the conversation.
